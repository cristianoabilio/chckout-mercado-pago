
## Checkout integrated to Mercado Pago Gayments Gateway

Project built using: Laravel 12, PHP 8.2, Mysql, Livewire, Inertia, Sail, Mail Pint,Sanctum, Breeze, Mercado Pago SDK and Pest tests.

Some steps of development:

- Modeling: Database, Migrations, Seeds, Factories, Enums, and Models.
- Checkout in Livewire - Personal Infos, Payment (credit card, Pix and bank slip), and shipping connecting to Via Cep API.
- Backend: Connecting to Mercado Pago API, Creating and updating card and order, Create User, Payment according to the chosen method, handle with exceptions, getting the address by zipcode by Via Cep API, Sending emails using queues, signed routes to success page, Webhooks, Magic Login, Pest Tests.

Laravel is accessible, powerful, and provides tools required for large, robust applications.
